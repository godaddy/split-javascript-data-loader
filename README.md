# split-javascript-data-loader

 Loads serialized rule/segment data from Split.io into the browser local storage.

## Installation

Install via npm:

```console
$ npm i @godaddy/split-data-loader --save
```

## Testing

Run the linter:

```console
$ npm run lint
```

Run unit tests:

```console
$ npm test
```

Generate a coverage report:

```console
$ npm run coverage
```
